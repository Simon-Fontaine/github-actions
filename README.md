# TP01 - Application de base

## 🎯 Objectif

Comprendre la structure d'une application Node.js simple avec des tests unitaires.

## 📦 Installation

```bash
npm install
```

## 🧪 Exécuter les tests

```bash
npm test
```

## 📚 Contenu

- `app.js` : Fonctions mathématiques simples
- `app.test.js` : Tests unitaires avec Jest
